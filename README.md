# 🌐 Shivam Sharma – Personal Portfolio Website

This is my personal developer portfolio built using Next.js and Tailwind CSS.  
It showcases my projects, technical skills, certifications, and contact information.

🔗 Live Website: https://my-portfolio-three-phi-63.vercel.app/

---

## 📌 About The Project

This portfolio was designed and developed to present my technical skills, full-stack development experience, and project work in a clean, modern, and professional format.

The goal of this project is to:

- Highlight my MERN stack expertise
- Showcase real-world projects
- Provide recruiters with easy access to my resume
- Maintain a strong professional online presence

---

## 🛠 Tech Stack

- Next.js (App Router)
- React.js
- Tailwind CSS
- Vercel (Deployment)

## ✨ Features

- Fully Responsive Design (Mobile + Desktop)
- Modern and Minimal UI

- Skills Section
- Contact Section
- Smooth Scrolling & Hover Effects
